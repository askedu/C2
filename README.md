# Введение

Работа направлена на изучение темы "Равновесие произвольной плоской системы сил". Предлагается решить задачу, из данного материала, самостоятельно и проверить результат решения с применением ЭВМ. После чего можно перейти к решению собственной задачи.

Для вычисления с применением ЭВМ используется язык программирования Python и проект Jupyter, которые являются общедоступными и не требуют установки. Предлагаемый вариант использования Python основан на Google Colaboratory, или просто Colab \([https://colab.research.google.com/](https://colab.research.google.com/)\), позволяет писать и выполнять код Python в браузере.
